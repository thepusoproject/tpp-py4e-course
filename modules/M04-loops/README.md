## 4) Lab — Feature: Multi-item cart
### User story
As a cashier, I want to add multiple items to a cart and see the final receipt.

### Requirements
- Repeatedly ask for: item name (or done), price, qty
- Keep running subtotal
- When done, print receipt: number of line items, subtotal
