## MODULE 00 — Setup + First Pull Request (Submission)

### Objective
By the end, kaya mo na:
- gumawa ng repo from template
- run the project locally
- gumawa ng branch + commit + push
- gumawa ng **Pull Request (PR)**

### Requirements
1) Create your own repo:
- Go to the template repo: **TPP Sari-Sari Store**
- Click **Use this template**
- Name it: `tpp-sari-sari-store-<yourname>`
- Set it to **Public**

2) Clone + run:
```bash
git clone <your-repo-url>
cd <your-repo-folder>
python main.py
```

3) Edit `STUDENT.md` and fill it in:
- Name
- Why you’re learning Python
- Weekly time budget
- Goal

4) Create a branch + commit + push:
```bash
git checkout -b module-00-setup
git add STUDENT.md
git commit -m "M00: add student profile"
git push -u origin module-00-setup
```

5) Open a Pull Request:
- PR Title: **Module 00 - Setup**
- PR Description must include:
  - screenshot (or pasted output) of `python main.py`
  - 2–3 sentences: what you installed / issues encountered
