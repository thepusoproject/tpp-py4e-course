# START HERE (TPP Py4E Taglish Track)

Welcome! This course is based on **Python for Everybody (Py4E)** and taught in **Taglish** (English keywords, Filipino explanation).

Your main project is a single evolving case study:

> **TPP Sari-Sari Store (CLI)**

## What you need
- Python 3 installed
- Git installed
- A GitHub account (your normal account is fine—inviting the instructor only exposes this repo. If you prefer extra separation, feel free to create a dedicated account just for the course.)
- (Recommended) VS Code + Python extension

## How the course works (self-paced)
- You work **module by module**
- Unlock rule: You need **80%** to unlock the next module.

## Step 1 — Create your project repo
1) Go to the template repo: **TPP Sari-Sari Store**
2) Click **Use this template**
3) Name your repo: `tpp-sari-sari-store-<yourname>`
4) Set it to **Public** (unless instructed otherwise)

## Step 2 — Clone and run
```bash
git clone <your-repo-url>
cd <your-repo-folder>
python main.py
```

## Step 3 — Submit each module
- Create a branch
- Commit + push
- Open a PR
